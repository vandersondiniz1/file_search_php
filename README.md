#FILE_SEARCH_PHP
Sistema para buscar arquivos em diretórios e sub-diretórios.

Frequentemente, havia chamados para busca de áudios de ligações telefônicas antigas que serviriam para a defesa da empresa junto a contestações judiciais. 
Em um dos servidores da empresa, há áudios do período de 2016 ao corrente ano (2021). Esse trabalho, dado o caráter mais técnico, era sempre realizado pela equipe de TI que manualmente tinha que procurar os áudios, seja via explorer do Windows, seja via comandos no terminal Linux.

O sistema foi criado para suprir a necessidade de eliminar trabalhos manuais que envolvem o setor de TI, assim como dar uma resposta mais rápida aos chamados abertos pela, em sua maioria, equipe de monitoria. Chamados relacionados à busca de áudio normalmente são abertos apenas por este setor.
